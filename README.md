# SRT — Semantic Reattribution Theory

**A provisional framework for value-attribution auditing in reception, community generation, and platform amplification.**

**受容・共同体生成・プラットフォーム増幅における価値帰属監査のための暫定理論枠組み。**

---

## Core Proposition / 核命題

> That something has value and whose work that value should be credited to are separate questions.

> 価値があることと、その価値を誰の仕事として計上するかは別問題である。

---

## What is SRT? / SRT とは何か

SRT is an operationally grounded value-attribution auditing framework. It audits the structure by which meaning, value, and richness generated or amplified through reception, community, and platform processes come to be recounted as the intrinsic achievement of works, agents, or theories themselves.

SRT は、操作的基盤を持った価値帰属監査フレームワークである。受容・共同体・プラットフォームの過程で生成または増幅された意味・価値・厚みが、作品・主体・理論それ自体の固有達成として再計上される構造を監査する。

SRT does not deny value itself. It audits the confusion between where value originates and to whom it is attributed.

SRT は価値そのものを否定しない。価値の発生地点と帰属先の混線を監査する。

## What's New in v0.7 / v0.7 の変更点

v0.7 adds external-operation collapse countermeasures while preserving the theoretical core of v0.6. Key additions:

v0.7 は v0.6 の理論核を保持したまま、外部適用崩壊リスクへの対策を追加した。主な変更点：

- NOT-Boundaries expanded with external-operation boundary items / NOT 境界に外部運用上の境界項目を追加
- External Operation Conditions (§11) added, delegating details to a separate spec / 外部運用条件（§11）を新設し、詳細を別紙に委任
- "Internal Anchor" renamed to "Primary-Object Anchor" for precision / 「内部アンカー」を「一次対象アンカー」に改称
- Self-Auditability (§12) expanded to cover vocabulary-circulation risk / 自己監査可能性（§12）に語彙流通リスクを追加
- External Operation Spec introduced: state categories, pre-application gates, invalid-application criteria, fixed auditing format / 外部運用規約を新設：状態区分、適用前ゲート、失格条件、固定監査フォーマット

## Status / 位置づけ

SRT v0.7 is a provisional framework equipped with a theoretical core, the skeleton of operational procedures, and explicit external-operation conditions. It is not yet a completed theory. Open Checks are listed for both the Core Text (16 items) and the External Operation Spec (6 items).

SRT v0.7 は、理論核と操作手続きの骨格に加え、外部運用条件を明示した暫定フレームワークである。完成理論ではない。Core 本文（16項目）と外部運用規約（6項目）の両方に Open Checks が明示されている。
